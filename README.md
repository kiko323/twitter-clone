# Twitter-clone
Features include:
- Guests can post
- Guests can see all posts
- Built-in administrator account
- "Administrator" can delete posts


## installing the project

After cloning the repository, run:
```
composer install
```

## MYSQL atabase
Import the .sql file, and edit the .env file accordingly.
